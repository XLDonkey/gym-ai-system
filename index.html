// What Is This

A gym that watches itself

The XL Fitness AI Overseer is a live intelligence layer overlaid onto the physical gym floor. A network of ceiling-mounted 4K cameras feeds a real-time AI pipeline that tracks every member, counts every rep, and logs every set — automatically.

No phones. No QR codes. No manual input. A member walks in, does their workout, and walks out. That evening they open the app and see a complete log of every exercise, every set, every rep, and every weight — captured automatically from above.

The Bible is the complete technical blueprint for how this system is built — from the first camera on the ceiling to the production-grade architecture that scales it to hundreds of gyms.

📷
4K Ceiling Camera Network
PoE dome cameras on a suspended steel grid frame. One Cat6 cable per camera, covering every machine from directly overhead.
🧠
Three Neural Networks
PoseNet (state classification), FaceNet (member identity), and WeightNet (weight detection) — running in parallel on each Raspberry Pi via ONNX Runtime.
🚦
The State Gate
A priority mutex that enforces exactly one output state at a time — preventing the AI from believing a person is both mid-rep and resting simultaneously.
🔄
Self-Improving via Active Learning
When the model is uncertain, it uploads the clip for human review. Annotations feed back into training. The system gets smarter with every gym session.

// The 46ms Pipeline

From pixel to workout data

Seven stages, 46 milliseconds. Every frame processed in real time, every rep counted automatically.

1
Camera Capture
4K PoE dome cameras stream H.265 video over Cat6 cable to the server at 30 FPS
2ms
2
YOLO Detection
YOLOv8 scans each frame and draws a bounding box around every person in the scene
8ms
3
DeepSORT Tracking
Assigns a persistent ID to each person and tracks them as they move between machines
4ms
4
MoveNet Pose Estimation
Maps 17 body joints for each tracked person, enabling joint angle calculations
12ms
5
LSTM Rep Counter
Analyses the sequence of joint positions over time to detect completed reps
3ms
6
Face ID Match
Extracts a face embedding and matches it to a registered gym member's profile
6ms
7
Database Write
Logs the completed set — member, machine, reps, weight — to the PostgreSQL database
9ms
